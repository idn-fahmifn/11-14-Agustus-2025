Ini adalah elemen display block Display block akan menguasai 1 baris penuh di setiap elemennya.

Display block

  1. ukuran bisa dicustom
  2. menguasai 1 baris
  3. berlaku padding dan margin
Ini adalah display inline Inline akan mengambil 1 baris dan ukurannya sesuai konten
Display Inline
  1. ukuran tidak bisa diatur, akan mengikuti ukuran dari kontennya
  2. padding berlaku, margin hanya sumbu-X (kiri dan kanan)
Ini adalah inline block Menguasai 1 baris

Display Inline :

  1. menguasai 1 baris seperti inline
  2. bisa ubah width dan height
  3. padding dan margin berlaku
